Key Learnings of Video Make Benefit Glorious Homework Assignment

  • During KiCAD Import, use Edge Cuts layer.
  • Set an origin with Place Auxilliary Axis Origin.
  • You can import multiple DWG's into one KiCAD project.
  • You can draw out the FR1 outline so you know how many parts you can lay up at once.
  • Use File > Plot to export.
    • Uncheck all but Edge Cuts.
    • Pick any output directory.
    • Use relative paths.
    • Click Plot.
  • When importing into Bantam Tools you need to do a hack.
    • Set Outline> to the gerber file.
    • Ignore the Top value. Bantam needs it to be there, so accept it and be tolerant.
  • There's also some normal Bantam Tool settings to fiddle with.
    • Uncheck Traces.
    • Material: Generic.
    • Measure youre material thinkness.
    • Placement Z = material + double-side tape. Measure a single; make a tape sandwich, measure the sandwich, subtract.
    • Probalby gonna stick with 1/32 in end mill.
    • In OnShape, use fillete. Make "notch length" short enough to accomodate them.

Mke a Spar

This should be as easy as taking Joshua's box-side and shrinking a dimension or two. But first, some measurements.

  • Joshua's slot length is 26 mm from one arc-center to the next.
  • Joshua's notch width is 24 mm across, not counting the fillets.
  • Joshua's fillets are 1 mm. Looks like these will come free if I copy his part.

The key thing here, is that the slot is 2 mm longer than the notch. So we probably wanna 4 mm slot with a 2mm notch. Then, we gotta ...

  1. Start with Joshua's box side
  2. reduce slot halfwidth from 12 to 1
  3. reduce overall side with from NN to YY

Turns out all the sketches and dimensions in Joshua's box are inter-realted to each other. First, shrink his slot lenghts on his Sketch 1down to 4mm.

Next, Fiddle with Sketch1 side lenth and slot-to-edge distance to make the smallest box possible. Baslically side length can go down to 9mm and slot-to-edge can go down to 1.57/2 + 0.1 before anything breaks too badly.

Moving on to box sides (Sketch2). Shrink the noth half-length from 12 mm down to 2. The rest of the side got "skinny" after shrinking Sketch1.

Ah, Crap

It didn't cut all the way through.

The parts came out with some persuasion and a utility knife. It looks like my notch and slot will work if properly milled. Okay, so last time I told Bantam my Material had a Z Offset of 1.56 mm, causing the mill to stop before cutting all the way through. So if I tell Bantam my material is lower it should cut deeper. It's pretty hard to get a caliper reading, on the layer that was left behind. I'm getting something like 0.34 mm. I want this to go through, so Imma round up to 0.4mm, bringing my new Materia Z Offset to 1.1mm
