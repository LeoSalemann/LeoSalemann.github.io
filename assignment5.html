DAY TWO: Sketch It! Sketch It Good!

Took a day off yesterday, got some ideas on how to reuse/repurpose the in-class box demo. It's all about slots and notches. I can reuse the slots in my airfoils, and the notches in some between-foil spars.

I'll probalby need to do the same "notch into notch" technique I used in my carboard wings. The key dimension here is slot width w (below). If I can assume that the in-class box demo will indeed product a snug-fitting box, then the widths of the slots should be the w I need.

A little poking around with the OnShape file tells me w = 1.6mm

Okay, so to do this, Imma need to:

  1. Start with Joshua's in-class box.
  2. Grab a side, make it skinny so it becomes a spar.
  3. Grab a slot, shorten it so it can be used in an airfoil.
  4. Make some spars with my HW3 Grashopper files. No fillets in Grashopper; add them in OnShape.
  5. Make an airfoil with slots in Grashopper, also no fillets.
  6. Harmonize all the geometry in OnShape.
  7. Run the OnShape -> Bantam workflow.

Key Learnings of Video Make Benefit Glorious Homework Assignment

  • During KiCAD Import, use Edge Cuts layer.
  • Set an origin with Place Auxilliary Axis Origin.
  • You can import multiple DWG's into one KiCAD project.
  • You can draw out the FR1 outline so you know how many parts you can lay up at once.
  • Use File > Plot to export.
    • Uncheck all but Edge Cuts.
    • Pick any output directory.
    • Use relative paths.
    • Click Plot.
  • When importing into Bantam Tools you need to do a hack.
    • Set Outline> to the gerber file.
    • Ignore the Top value. Bantam needs it to be there, so accept it and be tolerant.
  • There's also some normal Bantam Tool settings to fiddle with.
    • Uncheck Traces.
    • Material: Generic.
    • Measure youre material thinkness.
    • Placement Z = material + double-side tape. Measure a single; make a tape sandwich, measure the sandwich, subtract.
    • Probalby gonna stick with 1/32 in end mill.
    • In OnShape, use fillete. Make "notch length" short enough to accomodate them.
