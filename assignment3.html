Export to STL, pull into Cotangent. Hmm, this time, it didn't prompt me to convert to inches. Back to Rhino, set both Layout Units and Model Units to inches. Export to STL, load directly into Dremel DigiLab. Doesn't help. More scale-hacking, this seems to have worked:

  1. Original model was build in inches, while Rhino was set to milimeters
  2. Go to the "Document Properties" toolbar icon, set both Layout Units and Model Units to inches.
  3. Go to the "file" toolbar icon, set both Model Units back to milimeters.
  4. When prompted to scale model by 25.4, click Yes.
  5. Export to STL.
  6. Load into Dremel Digilab ... looks right
